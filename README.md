# Todo-list-app
Feature-rich Todo List app built with vanilla Javascript and LocalStroage
This project allows users to manage their daily tasks with persistent storage using localStorage.
## Features 
-Add new tasks 
-Select a due date for each task
-Button text changes to "Added!" after adding a task
-Mark tasks as completed 
-Completed tasks become visually crossed out
-Button text changes to "Completed!" after completing a task
-Delete individual tasks
-Clear all tasks at once
-Data persists after page reload (localStorage)
-Hover effects for better UI experience 
-Press 'Enter' to quickly add a new task
## Data Persistance
All tasks and their selected dates are saved in the browser using LocalStorage, so they remain even after refreshing the page.
## How to run
1) Download or clone the repository
2) Open 'todo-list.html' in your browser
